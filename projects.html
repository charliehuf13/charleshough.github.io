🧬 Large-Scale Phytophthora Comparative Genomics Pipeline

Ongoing Fall 2024 - Present Chang Lab Research

Project Scope: Leading bioinformatics analysis for a large-scale departmental Phytophthora genomics project, processing 108 oomycete isolates with primary, hap1, and hap2 genome assemblies to generate comprehensive synteny comparisons and evolutionary insights.

Technical Implementation:

  • Genome Processing: Managing 324 total assemblies (108 × 3 haplotypes) with decontaminated FASTA files and Helixer ML annotations
  • Automated Workflows: Developed SLURM-based pipeline using AGAT for GFF3-to-CDS conversion across all isolates
  • Synteny Analysis: Implementing GENESPACE comparisons for 6 comparison types per isolate (1,944 total analyses)
  • HPC Management: Orchestrating computational workflows on OSU's CQLS cluster with Singularity containers
  • Data Architecture: Designed hierarchical directory structure for organized output management

Key Workflow Components:

  • Step 1: Batch CDS extraction using agat_sp_extract_sequences.pl with quality filtering
  • Step 2: Automated R script generation for GENESPACE comparative analyses
  • Step 3: Synteny plotting for primary vs hap1/hap2, haplotype comparisons, and self-comparisons
  • Developed bash scripts for array job submission managing 648 comparison scripts
GENESPACE AGAT Helixer SLURM Singularity R Bash Scripting HPC
Impact: Developing scalable bioinformatics infrastructure for comparative genomics, contributing to understanding of oomycete genome evolution and pathogen diversity
